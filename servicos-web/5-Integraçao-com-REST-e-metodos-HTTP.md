# Introdução com REST e métodos HTTP



### Código de Estado

- Usado pelo servidor para avisar o cliente sobre o estado da operação solicitada.

- 1xx - Informativo.

- 2xx - Sucesso

- 3xx - Redirecionamento.

- 4xx - Erro do Cliente.

- 5xx - Erro do Servidor.
